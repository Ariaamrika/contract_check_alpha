# Contract Check Alpha

This repository contains a project named **Contract Check Alpha** that generates an OpenAI assistant for contract checking. The Streamlit application allows users to upload their contracts and receive responses from the assistant. The assistant leverages several technologies and tools, including OpenAI functions and file search tools.

## Features

- Contract uploading and checking via Streamlit interface
- Integration with OpenAI assistant for contract analysis
- Utilizes various libraries for processing and analysis
